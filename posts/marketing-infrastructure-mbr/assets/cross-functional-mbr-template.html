Section Template

Standardized Structure

Every section followed the same six-part structure. This standardization was the behavioral intervention to prompt analytical reporting from teams that had only done data entry.

Component 01
Owner & Description
Named functional owner responsible for data accuracy and narrative. Brief description of what the section covers.
Component 02
Metrics Table
Monthly columns with month-over-month, year-over-year and year-to-date calculations. Metrics specific to each function (sessions, orders, revenue, CTR, subscribers, etc.).
Component 03
Supporting Dashboards
Direct links to the live data sources. Looker Studio boards, Google Sheets trackers, Monday.com boards, platform-specific dashboards. The reference layer that made verification possible.
Component 04
Definitions
Standardized term definitions specific to each function. What counts as a "session," "new customer," "repeat customer," "engaged session." The shared vocabulary that made cross-functional comparison possible.
Component 05
Performance Summary
Narrative analysis written by the section owner. Not a restatement but an interpretation. What happened, why and what it means for the next period.
Component 06
Upcoming Initiatives
Forward-looking section listing planned work, in-progress items and dependencies. Created visibility across teams that had never shared roadmaps.
